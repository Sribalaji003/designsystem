Timing Tokens
Token Value Use Case
--dur-fast150msTap, press, hover state changes
--dur-med280msCard transitions, chip activate
--dur-slow450msPage transitions, expand/collapse
--ease-springcubic-bezier(0.34,1.56,0.64,1)Buttons, FAB, card hover
--ease-smoothcubic-bezier(0.4,0,0.2,1)Color transitions, opacity
